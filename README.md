Projeto no android studio usando kotlin para a faculdade
Primeira tentativa de um projeto que usa a camera para detectar o objeto na tela e gerar informações sobre ele 

Funcionalidade:
Analisar imagens com ia identificar oque é  o produto, gerar informações sobre e sugerir preços e onde comprar o objeto

Tecnologias sendo utilizadas:
Kotlin: (https://kotlinlang.org/)
Jetpack Compose: (https://developer.android.com/jetpack/compose)
CameraX: (https://developer.android.com/training/camerax)
Google Generative AI SDK: (https://ai.google.dev/)
